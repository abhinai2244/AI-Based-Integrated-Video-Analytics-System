Glossary of Key Terms
Term Definition
mAP Mean Average Precision — standard metric for object detection model accuracy across IoU thresholds.
YOLOv8 You Only Look Once v8 — state-of-the-art real-time object detection model by Ultralytics.
ANPR Automatic Number Plate Recognition — system for detecting and reading vehicle license plates.
OCR Optical Character Recognition — technology to convert images of text into machine-readable text.
DeepFace Lightweight Python framework for facial recognition and attribute analysis (age, gender, emotion).
FPS Frames Per Second — measurement of video processing speed and real-time capability.
Edge Computing Processing data locally on devices near the source, reducing latency and bandwidth usage.
Model Quantization Technique to reduce model size and increase inference speed by using lower precision (INT8).
Federated Learning Machine learning approach where models train on decentralized data without sharing raw data.
GDPR General Data Protection Regulation — EU regulation on data protection and privacy.
CCPA California Consumer Privacy Act — state-level data privacy law for California residents.
IoU Intersection over Union — metric measuring overlap between predicted and ground-truth bounding boxes.
